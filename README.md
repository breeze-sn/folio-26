# folio-26

Under-construction portfolio website built with React and Vite.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy on Render

1. Push this project to GitHub.
2. In Render, click **New +** > **Static Site**.
3. Connect the GitHub repo: `breeze-sn/folio-26`.
4. Use these settings:
   - Build Command: `npm install && npm run build`
   - Publish Directory: `dist`
5. Click **Create Static Site**.

Render auto-deploys on every push to the selected branch.
